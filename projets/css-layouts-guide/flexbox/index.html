Introduction à CSS Flexbox

CSS Flexbox (Flexible Box Layout) est un système de mise en page unidimensionnel qui permet de distribuer l'espace et d'aligner des éléments dans un conteneur.

Concept de base

1
2
3
.container {
  display: flex;
}

.item {
  flex: 1;
  padding: 1rem;
  background: #4a90e2;
  color: white;
  text-align: center;
}
<div class="container">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
</div>

Terminologie Flexbox

Flex Container

L'élément parent avec display: flex

Flex Items

Les enfants directs du conteneur flex

Main Axis

L'axe principal (horizontal par défaut)

Cross Axis

L'axe perpendiculaire au main axis

Propriétés des Éléments Flex

flex-grow

Détermine comment un élément grandit pour occuper l'espace disponible

0 1 0
1
2
3
.container {
  display: flex;
}

.item1 { flex-grow: 0; }
.item2 { flex-grow: 1; }
.item3 { flex-grow: 0; }
<div class="container">
  <div class="item item1">1</div>
  <div class="item item2">2</div>
  <div class="item item3">3</div>
</div>

align-self

Permet à un élément d'override l'alignement du conteneur

1
2
3
.container {
  display: flex;
  align-items: flex-start;
  min-height: 150px;
}

.special-item {
  align-self: auto;
}
<div class="container">
  <div class="item">1</div>
  <div class="item special-item">2</div>
  <div class="item">3</div>
</div>

Sidebar + Contenu

Sidebar
Contenu principal
.container {
  display: flex;
  min-height: 120px;
}

.sidebar {
  flex: 0 0 200px; /* Ne grandit pas, ne rétrécit pas, largeur fixe 200px */
  background: #7f8c8d;
  color: white;
  padding: 1rem;
  display: flex;
  align-items: center;
  justify-content: center;
}

.content {
  flex: 1; /* Prend tout l'espace restant */
  background: #4a90e2;
  color: white;
  padding: 1rem;
  display: flex;
  align-items: center;
  justify-content: center;
}
<div class="container">
  <div class="sidebar">Sidebar</div>
  <div class="content">Contenu principal</div>
</div>
