Organisation

Chaque démo reproduit un exemple de la fiche. Le CSS est centralisé dans guide.css.

  • A — fr : répartition de l’espace restant avec fr.
  • B — fixe + fr + gap : mix pistes fixes et fluides.
  • C — gap : espacement unique ou distinct (lignes/colonnes).
  • D — minmax : contraintes min/max par piste.
  • E — fit-content : colonnes limitées par le contenu.
  • F — min/max-content : largeur intrinsèque minimale vs maximale.
  • G — template-areas : layout déclaratif par zones.
  • H — auto-fit vs auto-fill : galerie responsive.
  • I — auto-flow: column : auto-placement vertical, colonnes implicites.
  • J — alignements : justify/align items / self / content.

Conseils de lecture

Ouvrez chaque page, lisez le cartouche d’explication puis redimensionnez la fenêtre pour observer le comportement.

Toutes les couleurs sont sobres pour une bonne lisibilité ; adaptez-les à votre charte si besoin.
